In the project directory, you can run:

### `npm run dev`
Run both client-side and server-side concurrently.

### `npm run server-dev`
Run the server under nodemon.

### `npm run start` 
Run React development mode.

### `npm run build`
Run React build command.

### `npm install`
Install all dependencies.
